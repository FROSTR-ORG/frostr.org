# frostr.org

Marketing site for [FROSTR](https://frostr.org) — t-of-n remote signing and key rotation for Nostr.

Built with Vite, React, and TypeScript. Content and styling are ported incrementally from `frostr-website` and the [Paper design system](https://app.paper.design/file/01KS3AE1V20271ZW6FDG71S330).

## Development

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Typecheck and production build |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build |

See [AGENTS.md](./AGENTS.md) for project conventions.
